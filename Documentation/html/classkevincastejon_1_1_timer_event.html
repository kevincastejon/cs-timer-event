Detailed Description

An TimerEvent object is dispatched whenever a Timer event occurs.

These are all the available names for this event:

  • TIMER - Dispatched by a Timer instance each time the delay has ended
  • TIMER_COMPLETE - Dispatched each time the RepeatCount has reached the Repeat property on a Timer instance
